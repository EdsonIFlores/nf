# Manual do Cliente - FiscalFlow

O FiscalFlow organiza PDFs e XMLs de notas fiscais, boletos e documentos recebidos por e-mail ou adicionados manualmente.

## 1. Configurar o e-mail

1. Abra a aba **Configurar e-mail**.
2. Escolha o provedor.
3. Informe o e-mail completo.
4. Informe a senha de app.
5. Para Zoho com domínio próprio, use **Zoho domínio próprio (imappro)**.
6. Clique em **Testar acesso**.
7. Se funcionar, clique em **Salvar configuração**.

Para Gmail e Zoho, normalmente é necessário usar senha de aplicativo, não a senha normal da conta.

## 2. Configurar a pasta de salvamento

1. Abra a aba **Salvamento**.
2. Clique em **Escolher pasta**.
3. Escolha uma pasta local, Google Drive ou OneDrive sincronizado.
4. Clique em **Salvar destino**.

Quando um arquivo for marcado como **Conferido**, o sistema tentará salvar uma cópia nessa pasta.

## 3. Buscar notas fiscais

1. Abra a aba **Início**.
2. Clique em **Buscar agora** para buscar anexos no e-mail salvo.
3. Também é possível ativar **Buscar automaticamente enquanto o sistema estiver aberto**.
4. Para adicionar arquivos manualmente, use **Escolher arquivos** ou **Escolher pasta**.

## 4. Conferir documentos

1. Clique em um arquivo na tabela.
2. Use **Abrir arquivo** para visualizar.
3. Ajuste fornecedor, período, categoria, documento, CNPJ, chave e observações.
4. Mude o status para **Conferido**.

## 5. Pastas

Na aba **Pastas**, o sistema mostra as pastas criadas por fornecedor e período.

Use **Baixar pasta** para baixar os arquivos daquela pasta em ZIP.

## 6. Backup

Na aba **Backup**, é possível:

- baixar planilha CSV;
- baixar pacote Vobi;
- baixar backup JSON;
- restaurar backup JSON;
- copiar plano de pastas.

Faça backup periodicamente, principalmente antes de trocar de computador ou limpar o navegador.

## 7. Suporte

Na aba **Suporte**, clique em **Copiar diagnóstico** e envie pelo WhatsApp de suporte quando precisar de ajuda.
