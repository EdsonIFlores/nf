# Checklist de Entrega ao Cliente - FiscalFlow

Use esta lista antes de considerar a instalação entregue.

## Acesso e visual

- [ ] Abrir o link do sistema.
- [ ] Confirmar se o logo TEC-PRO carregou.
- [ ] Confirmar se o botão **Suporte WhatsApp** abre com mensagem pronta.
- [ ] Verificar se as abas aparecem corretamente.

## E-mail

- [ ] Configurar Gmail com senha de app.
- [ ] Clicar em **Testar acesso**.
- [ ] Buscar PDFs e XMLs.
- [ ] Configurar Zoho com `imappro.zoho.com`, porta `993`, SSL ativo.
- [ ] Testar busca em `INBOX`.
- [ ] Confirmar que a pasta `Conversation History` não é usada para notas fiscais.

## Arquivos

- [ ] Adicionar PDFs manualmente.
- [ ] Adicionar XMLs manualmente.
- [ ] Importar anexos pelo e-mail.
- [ ] Abrir um PDF pelo botão **Abrir arquivo**.
- [ ] Abrir um XML pelo botão **Abrir arquivo**.
- [ ] Editar fornecedor, período, CNPJ, chave e observações.
- [ ] Marcar documento como **Conferido**.

## Salvamento

- [ ] Escolher pasta local ou pasta sincronizada do Drive/OneDrive.
- [ ] Marcar um arquivo como **Conferido**.
- [ ] Confirmar se o arquivo foi copiado para a pasta escolhida.

## Exportação e backup

- [ ] Baixar planilha CSV.
- [ ] Baixar pacote Vobi.
- [ ] Baixar pasta ZIP.
- [ ] Baixar backup JSON.
- [ ] Restaurar backup JSON.
- [ ] Confirmar que os registros voltaram corretamente.

## Suporte

- [ ] Abrir aba **Suporte**.
- [ ] Clicar em **Copiar diagnóstico**.
- [ ] Colar o diagnóstico em uma conversa de teste.

## Observações

Para a primeira entrega comercial, combine com o cliente:

- qual e-mail será usado;
- quem terá acesso à senha de app;
- onde os arquivos conferidos serão salvos;
- frequência de suporte;
- valor mensal de manutenção.
